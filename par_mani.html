Programmēšanas valodas

Manas programmēšanas valodas

Šeit jūs varēsiet redzēt kādas un cik daudz programmēšanas valodas esmu apguvis.

Python

Python ir augsta līmeņa, vispārējas nozīmes programmēšanas valoda. Tā atbalsta vairākas programmēšanas paradigmas, tostarp strukturētu, objektorientētu un funkcionālu programmēšanu. Gvido van Rosums sāka strādāt ar Python 80. gadu beigās kā ABC programmēšanas valodas pēcteci un pirmo reizi to izlaida 1991. gadā kā Python 0.9.0. Python 2.0 tika izlaists 2000. gadā, un tajā tika ieviestas jaunas funkcijas, piemēram, saraksta izpratne, cikla noteikšanas atkritumu savākšana un Unikoda atbalsts. Python 3.0 bija liela versija, kas nav pilnībā saderīga ar iepriekšējām versijām.

C#

C# ir vispārēja lietojuma, vairāku paradigmu programmēšanas valoda. C# ietver statisko mašīnrakstīšanu, stingru mašīnrakstīšanu, leksikas darbības jomu, imperatīvu, deklaratīvu, funkcionālu, vispārīgu, objektorientētu un uz komponentiem orientētas programmēšanas disciplīnas. C# programmēšanas valodu izstrādāja Anders Hejlsberg no Microsoft 2000. gadā, un vēlāk kā starptautisku standartu to apstiprināja Ecma 2002. gadā un ISO/IEC 2003. gadā. Microsoft ieviesa C# kopā ar .NET. Framework un Visual Studio, kas abi bija slēgtā koda. Tajā laikā Microsoft nebija atvērtā pirmkoda produktu.

C++

C++ ir vispārējas nozīmes programmēšanas valoda, ko radījis dāņu datorzinātnieks Bjarne Stroustrup kā C programmēšanas valodas paplašinājumu. Laika gaitā valoda ir paplašinājusies, un modernajam C++ tagad ir vispārīgas un funkcionālas funkcijas papildus zema līmeņa atmiņas manipulācijas iespējām. Tā gandrīz vienmēr tiek ieviesta kā kompilētā valoda, un daudzi pārdevēji nodrošina C++ kompilatorus, tāpēc tas ir pieejams daudzās platformās. Programma C++ tika izstrādāta, ņemot vērā sistēmu programmēšanu un tās dizaina galvenās iezīmes ir veiktspēja, efektivitāte un izmantošanas elastība.

HTML

HTML ir iezīmēšanas valoda, kas ir vērsta uz vienas lappuses teksta dokumentu attēlošanu speciālās programmas HTML lietotāja aģentos. HTML sniedz iespēju dokumenta pamatsaturu sasaistīt ar dažāda veida metadatiem nepieciešamo informāciju. Atveides informācija var būt gan nelielas teksta dekorācijas, gan sarežģīti skripti kas kontrolē tīmekļa pārlūkprogrammu. Metadati var saturēt informāciju par dokumenta nosaukumu un autoru, izteikt to, kā dokuments tiek dalīts rindkopās, sarakstos un kas ir pats svarīgākais, informāciju, kas atļauj dokumentu sasaistīt ar citiem dokumentiem.

CSS

CSS jeb Stila lapas kaskadēšana ir īpaša stila lapas valoda, ko lieto, lai aprakstītu izskatu iezīmēšanas valodā veidotiem dokumentiem. Populārākais pielietojums ir HTML un XHTML valodās veidotu tīmekļa lappušu izskata aprakstīšanai, bet to var izmantot jebkādiem XML dokumentiem. Par dokumenta stilu šajā gadījumā saprot tā elementu īpašības, kas raksturo ārējo izskatu. CSS tika iecerēts, lai atdalītu dokumenta saturu no tā stila. Tas ļauj dokumenta izskatu pielāgot konkrēta lietotāja vajadzībām. Bez tam vienam dokumentam var pielietot vairākus stilus, piemēram, vienu skatīšanai uz ekrāna, otru izdrukāšanai uz papīra.

JavaScript

JavaScript ir programmēšanas valoda, kas ir viena no globālā tīmekļa galvenajām tehnoloģijām līdzās HTML un CSS. No 2022. gada 98% vietņu tīmekļa lapu darbībai klienta pusē izmanto JavaScript. Visām lielākajām tīmekļa pārlūkprogrammām ir speciāls JavaScript dzinējs, lai izpildītu kodu lietotāju ierīcēs. JavaScript ir augsta līmeņa, kas atbilst ECMAScript standartam. Tā ir vairāku paradigmu, kas atbalsta uz notikumiem orientētus, funkcionālus un obligātus programmēšanas stilus. Tam ir lietojumprogrammu saskarnes darbam ar tekstu, datumiem, regulārām izteiksmēm, standarta datu struktūrām.
